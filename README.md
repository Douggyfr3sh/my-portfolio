# Software Portfolio

A simple, responsive HTML portfolio website built with Bootstrap 5 to showcase software development skills, experience, and projects.

## Features

- **Responsive Design**: Built with Bootstrap 5 for mobile-first responsive layout
- **Clean Navigation**: Top navbar with centered title and tabbed navigation for different sections
- **About Me Section**: Profile image area and introductory text with professional description
- **Tabbed Content Areas**:
  - Work History
  - Education
  - Portfolio Projects
  - Work Style
  - Hobbies
  - Personal Info

## Technologies Used

- HTML5
- Bootstrap 5.3.2 (via CDN)
- CSS3 (minimal custom styling)

## Getting Started

1. Clone this repository
2. Open `index.html` in a web browser
3. Customize the content with your personal information

## Customization

### Profile Image
Replace the black placeholder box with your profile image:
- Update the CSS class `.profile-image-placeholder`
- Or replace with an `<img>` tag containing your photo

### Content Sections
Fill in the empty tab sections with your personal content:
- **Work History**: Add your professional experience
- **Education**: Include degrees, certifications, and courses
- **Portfolio**: Showcase your projects with descriptions and links
- **Work Style**: Describe your development approach and methodologies
- **Hobbies**: Share relevant personal interests
- **Personal Info**: Add contact information and availability

### Styling
- Update the navbar title with your actual name
- Modify the Bootstrap theme or add custom CSS in the `<style>` section
- Customize colors, fonts, and layout to match your personal brand
