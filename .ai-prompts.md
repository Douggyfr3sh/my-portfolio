# Portfolio Development Prompts

This file contains useful prompts for AI assistants when working on this software portfolio project.

## Content Development Prompts

### Work History Section
```
Help me create content for the Work History tab. I need to add my professional experience including:
- Company names and roles
- Employment dates
- Key responsibilities and achievements
- Technologies used
Format this using Bootstrap cards or timeline components.
```

### Portfolio Projects Section
```
I want to showcase my software projects in the Portfolio tab. Help me create:
- Project cards with descriptions
- Technology badges/tags
- Links to GitHub repositories or live demos
- Project screenshots or icons
Use Bootstrap card components with proper responsive layout.
```

### Skills and Technologies Section
```
Add a skills section to display my technical competencies:
- Programming languages
- Frameworks and libraries
- Tools and platforms
- Soft skills
Create a visually appealing layout using Bootstrap badges, progress bars, or icon grids.
```

## Design Enhancement Prompts

### Responsive Design
```
Review the current portfolio layout and ensure it's fully responsive:
- Test mobile, tablet, and desktop views
- Adjust image sizes and text layouts
- Optimize navigation for different screen sizes
- Ensure all Bootstrap components work properly on mobile
```

### Visual Improvements
```
Enhance the visual design of the portfolio:
- Add a color scheme that's professional but engaging
- Implement subtle animations or transitions
- Add icons to the navigation tabs
- Improve typography and spacing
- Consider adding a hero section or call-to-action
```

### Accessibility
```
Make the portfolio website more accessible:
- Add proper ARIA labels and roles
- Ensure keyboard navigation works correctly
- Check color contrast ratios
- Add alt text for images
- Test with screen readers
```
